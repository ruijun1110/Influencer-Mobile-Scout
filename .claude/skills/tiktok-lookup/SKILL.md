# TikTok Lookup — iMessage Bot

Manages a background daemon that watches iMessages for TikTok profile URLs and replies with similar creator links.

## How it works

1. Anyone sends your Mac's iMessage a TikTok profile URL (e.g. `tiktok.com/@someuser`)
2. Bot replies immediately: `Looking up similar creators for @someuser...`
3. Bot calls TikHub API → gets `sec_uid` → fetches similar users
4. Bot replies with up to 10 similar creator profile links
5. Results saved to `data/similar_users.xlsx`

## Commands

### `/tiktok-lookup start`

Check prerequisites, then start the daemon:

```bash
# 1. Verify uv is available
which uv || (echo "ERROR: uv not found. Run: bash setup.sh" && exit 1)

# 2. Verify TIKHUB_API_KEY is set
grep -q 'TIKHUB_API_KEY=.' ".claude/.env" 2>/dev/null || {
  echo "ERROR: TIKHUB_API_KEY not set in .claude/.env"
  exit 1
}

# 3. Verify plist exists (generated by setup.sh)
[ -f ".claude/skills/tiktok-lookup/launchd/com.tiktok-lookup.plist" ] || {
  echo "ERROR: plist not found. Run: bash setup.sh"
  exit 1
}

# 4. Copy plist to LaunchAgents and load
cp ".claude/skills/tiktok-lookup/launchd/com.tiktok-lookup.plist" \
   "$HOME/Library/LaunchAgents/com.tiktok-lookup.plist"

launchctl unload "$HOME/Library/LaunchAgents/com.tiktok-lookup.plist" 2>/dev/null || true
launchctl load "$HOME/Library/LaunchAgents/com.tiktok-lookup.plist"
```

Wait 3 seconds, then verify it started:

```bash
launchctl list | grep tiktok-lookup
```

If the PID column shows a number (not `-`), the daemon is running. Tell the user:
> "Bot is running. Send any iMessage containing a TikTok profile URL to test it."

If it shows `-` with a non-zero exit code, check the log:
```bash
tail -20 /tmp/tiktok-lookup.err
```
Report the error to the user.

### `/tiktok-lookup stop`

```bash
launchctl unload "$HOME/Library/LaunchAgents/com.tiktok-lookup.plist" 2>/dev/null
rm -f "$HOME/Library/LaunchAgents/com.tiktok-lookup.plist"
echo "Bot stopped."
```

### `/tiktok-lookup status`

```bash
launchctl list | grep tiktok-lookup && tail -5 /tmp/tiktok-lookup.log || echo "Bot is not running."
```

## Output

**xlsx schema** — `data/similar_users.xlsx`, sheet `Lookups`:

| queried_handle | similar_handle | profile_url | lookup_date | requested_by |
|---|---|---|---|---|

**iMessage reply format:**
```
Similar to @someuser:

@creator1
tiktok.com/@creator1

@creator2
tiktok.com/@creator2
...
(saved to similar_users.xlsx)
```

## Troubleshooting

| Error | Fix |
|---|---|
| `DATABASE` error in log | Grant Full Disk Access to Terminal: System Settings → Privacy & Security → Full Disk Access |
| `TIKHUB_API_KEY not set` | Add key to `.claude/.env`: `TIKHUB_API_KEY=your_key` |
| `uv: command not found` | Run `bash setup.sh` — it installs uv automatically |
| Bot not responding to messages | Ensure Messages.app is signed into iMessage; check `/tmp/tiktok-lookup.err` |
| PID shows `-` after load | Check `/tmp/tiktok-lookup.err` for startup crash details |
